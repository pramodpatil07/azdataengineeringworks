# azdataengineeringworks
Learning azure ADF and ADB

## Python utilities support

A new root folder `azpyworks` has been added for Python-based shared classes, configuration helpers, and notebook utilities.

- `azpyworks/config.py` - JSON config loader and utilities
- `azpyworks/notebook_utils.py` - notebook helper functions
- `azpyworks/configs/sample_config.json` - sample configuration file
- `azpyworks/README.md` - Python workspace usage guide

This folder supports Python notebooks alongside the existing Scala/Spark modules.
